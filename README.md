Требуется разработать второй простой асинхронный сервис на другом языке (Go) с одним http-методом для выполнения отложенного действия в вашей системе (вычисление). Действие выполняется с задержкой 5-10 секунд, результат сервиса случайный (например успех/неуспех, но записывается всегда) либо вычисляемый по формулам. Достаточно в результате обновить одно поле по предметной области в заявке/м-м. При создании заявки это поле пустое.
В исходном веб-сервисе также необходимо добавить http-метод для внесения результатов. Асинхронный сервис взаимодействует с основным через http, без прямого обращения в БД. Добавить псевдо авторизацию в методе основного сервиса - передавать как константу какой-нибудь ключ-токен, например на 8 байт, и через if просто проверять на совпадение это поле.
